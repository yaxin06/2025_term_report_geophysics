課程介紹

此課程分為兩大類：地球物理知識和資料處理技能。

上課方式

本課程結合線上學習與實地操作，學生將參與地震測量及數據分析工作坊。

成績計算

  • 實驗報告：40%
  • 期中考：30%
  • 期末專題報告：30%

折射探勘學

折射探勘學是研究地震波折射現象的學科，常用於研究地下結構。

校園野外實驗

這是一次地震波的折射探勘實驗，使用24個地震儀來記錄震波，計算岩層厚度。

數據分析

使用計算波速與折射波的數據來推算地層厚度。

甘禮有博士演講心得

講座中討論了地震學的前沿技術與實務應用，包括短期與長期監測的差異。

Hugging Face 互動程式

此工具可幫助您進行重力異常數據處理及可視化。
