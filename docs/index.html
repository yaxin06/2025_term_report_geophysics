折射探勘學與實作

深入探討彈性波在地下介質中的傳播規律，特別是臨界折射現象。我們不僅學習理論公式，更強調數位工具的應用。

🛠 核心工具鏈

  • ObsPy: 用於地震波形數據的讀取與預處理。
  • PyGMT: 繪製高品質的地震波形走時圖與地質剖面。
  • Gradio: 建立互動式網頁介面，即時展示數據分析結果。

📍 野外實驗重點

實地前往校園黑森林進行震測實驗，利用人工震源（鐵鎚敲擊鐵板）產生地震波，記錄並分析不同接收點的初至時間。

透視地球的聲音

重力探勘學

學習如何量測地球引力的微小差異，並將其轉化為對地下地質構造（如斷層、礦體）的理解。

Gravity Corrections in Python

# Free-Air Correction (FAC)

FAC = 0.3086 * height_meters

# Bouguer Correction (BC)

BC = 0.04193 * density_rho * height_meters

# Gravity Anomaly

Anomaly = G_obs - G_theory + FAC - BC
