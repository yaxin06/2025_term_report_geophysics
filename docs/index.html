火山與防災：黃有志博士

沉與浮的奧祕：火山渣與浮石

每一顆石頭都記錄了岩漿噴發那一瞬的壓力。火山渣氣孔連通，入水即沉；浮石氣孔獨立形成空氣室，因而能漂浮。

日本更利用地磁變化監測岩漿動態：當岩石加熱超過居禮溫度時會消磁，這成為預判岩漿上升的重要指標。
